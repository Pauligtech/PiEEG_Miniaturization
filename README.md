## Installation

- ```
  conda env create -f environment.yml
  conda activate moabb_model_optimization
  ```
- Place `MNE-fatigue-mi` folder inside `moabb_model_optimization` folder.
- Place `mne_data` folder in home folder (e.g., `~/mne_data`).
